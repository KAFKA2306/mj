# 麻雀における相手の読みと「癖」の心理学

## 概要
麻雀で相手を読むことは、ポーカーにおける「テル（癖）」を読むことに似ています。捨て牌（手出し vs ツモ切り）、間（ま）、反応を観察することで、相手の手牌に関する隠された情報を推測します。

## 1. 捨て牌分析（基本）

*   **ツモ切り（オンラインゲームではグレー表示など）**: ツモった牌をそのまま捨てること。手が進行していない、または変化していないことを示します。ツモ切りが長く続く場合、そのプレイヤーはすでにテンパイしているか、完全にオリている（ベタオリ）可能性が高いです。
*   **手出し**: 手牌の中から牌を選んで捨てること。手の構成が変化したことを示します。
    *   *安全牌 vs 危険牌*: 安全牌を手放して危険牌（生牌など）を残す場合、そのプレイヤーは前に出ている（攻めている）兆候です。
    *   *スライド*: ツモった牌と同じ牌を（手の中から）捨てること。安全確保や赤ドラの入れ替えなどを意味することが多いです。

## 2. 一般的な「癖」とパターン

*   **役牌の「間（ま）」**: 字牌（風牌/三元牌）を捨てる前に一瞬止まる場合、対子（トイツ）で持っていて、安全牌として残すか効率のために捨てるか迷っていた可能性があります。
*   **染め手の溢れ**: 2色の牌ばかり捨てている場合、残りの1色で染め手（ホンイツ/チンイツ）を作っている可能性が高いです。
*   **端牌（ヤオチュー牌）の切り出し**:
    *   1や9の早い切り出し: タンヤオ狙いのことが多いです。
    *   終盤での生牌（ションパイ）の字牌・端牌切り: テンパイ、あるいはテンパイに近い強いサインです。安全牌を処理して勝負に出ています。

## 3. 心理的な「テル」（リアル対局）

*   **理牌（リーパイ）**: 多くのプレイヤーは配牌後すぐに理牌します。理牌を止めた場合、テンパイした可能性があります。
*   **視線**: 捨て牌（河）を頻繁に見ている場合、リーチをかける前にフリテンでないか、あるいは安牌を確認している可能性があります。
*   **反応速度**: 即座にポンやチーが入る場合、鳴く形がすでに決まっていたことを示唆します。長い「間」のあとにスルーされた場合、鳴ける形（対子など）を持っていたが、あえて鳴かない判断をした可能性があります。

## 4. 攻略戦略

*   **壁（カベ）**: 8が4枚すべて見えている（捨て牌 + ドラ表示牌など）場合、7-8-9の順子は作れないため、9は（順子待ちに対して）安全です。
*   **スジの罠**: 上級者は、あなたがスジ（1-4-7の安全性）を信頼していることを知っています。あえて安全そうなスジに見せかけた単騎待ち（地獄単騎など）で罠を仕掛けることがあります。
*   **ダマテン（闇聴）**: 普段攻撃的なプレイヤーが急に静かになり、安全牌ばかり切っているように見える時、実は高打点の手でダマテンを入れて、ロンあがりを狙っている可能性があります。
