# 麻雀における相手の読みと技術論

## 概要
麻雀で相手の手牌を読む技術は、大きく分けて「捨て牌分析（理詰め）」と「心理的な読み（癖・テル）」の2つに分類されます。本ドキュメントでは、捨て牌から論理的に危険牌を推測する技術、副露（鳴き）読み、役への対応、そして最新のAI研究に基づく高度な戦術を解説します。

## 1. 捨て牌分析（基本）

相手の捨て牌（河）は、情報の宝庫です。最も重要なのは「手出し」と「ツモ切り」の違いを見極めることです。

*   **ツモ切り（暗い表示/グレー等）**:
    *   ツモった牌をそのまま捨てる行為。
    *   **意味**: 手牌が進行していない、または変化させる価値がない。
    *   **読み**: ツモ切りが長く続く場合、そのプレイヤーは「テンパイしている」か、完全にオリている「ベタオリ」状態の可能性が高いです。

*   **手出し（明るい表示/通常等）**:
    *   手牌の中から牌を選んで捨てる行為。
    *   **意味**: 手の構成が変化した、あるいは安全牌と入れ替えた。
    *   **読み**:
        *   **関連牌の所持**: 手出しされた牌の周辺は、手牌の中に持っていた可能性が高いです。また、その周辺が危険になる（または安全になる）ヒントになります。
        *   **安牌保持**: 危険な生牌（ションパイ）を残して安全牌を手出しした場合、攻撃態勢（イーシャンテンやテンパイ）であることを強く示唆します。
        *   **空切り**: テンパイしているが、手牌にある牌と同じ牌をツモった時にわざと手牌の方を切ることで、まだノーテンのように見せかける技術です。

## 2. 防御理論（守りのセオリー）

放銃（ロンされること）を避けるための論理的な防御技術です。

### 2.1 スジ（Suji）
両面待ちに対する安全度の理論です。相手が両面待ちの場合、特定の牌が通っていれば、その「スジ」にあたる牌も通る可能性が高いという考え方です。

*   **1-4-7の法則**: 「4」が捨てられていれば、相手は「23」や「56」の両面待ちで待てないため、「1」と「7」は両面待ちに対して安全です。
*   **2-5-8の法則**: 「5」が捨てられていれば、「2」と「8」は両面待ちに対して安全です。
*   **3-6-9の法則**: 「6」が捨てられていれば、「3」と「9」は両面待ちに対して安全です。

**注意**: スジはあくまで「両面待ち」に対する安全です。カンチャン、ペンチャン、単騎待ちには通用しませんが、確率は大幅に下がります。

### 2.2 カベ（Kabe - No Chance）
自分の手牌と見えている牌を利用して、相手が待ちを作れないことを証明する理論です。

*   **ノーチャンス**: ある数牌が場に4枚すべて見えている場合、その牌を使った順子（シュンツ）は作れません。
    *   例: 「8」が4枚見えている → 相手は「78」や「89」の形を持てない → 「9」で待つ（6789、789）ことが物理的に不可能 → **「9」は安全**。
*   **ワンチャンス**: 3枚見えている場合。残り1枚を相手が持っていなければ安全ですが、ノーチャンスほどの信頼度はありません。

### 2.3 外側（Sotogawa）
序盤に捨てられた数牌の外側（1寄りまたは9寄りの牌）は比較的安全というセオリーです。

*   例: 序盤（1段目）に「4」が捨てられている場合、その外側の「1」「2」「3」は比較的安全。
*   理由: 序盤に「4」を捨てるということは、「234」や「345」といった順子を作る意思がなかった（あるいはすでに足りている）可能性が高いため、その周辺で待っている可能性が下がります。

### 2.4 ベタオリの手順 (Betaori Algorithm)
放銃を回避するためのアルゴリズム的な手順。

1.  **現物 (Genbutsu)**: 100%安全な牌を切る。
2.  **合わせ打ち**: 直前のプレイヤーが切った牌を切る。
3.  **スジ・壁・ワンチャンス**: 確率的に安全な牌を探す。
4.  **暗刻の端牌**: 自分で3枚持っている1・9牌などは、シャンポン待ち以外に当たらないため比較的安全。
5.  **地獄待ち警戒**: 字牌単騎（特に地獄単騎）は、見た目枚数が少なくてもケアする必要があります。

### 2.5 応用防御技術
*   **回し打ち（Mawashi-uchi）**: 完全にオリるのではなく、安全な牌を切りつつ、テンパイの可能性を残す打ち方。
*   **合わせ打ち（Awase-uchi）**: 上家が切った牌と同じ牌を即座に切る。同巡内フリテンにより100%安全。

## 3. 待ち読み技術（一点読みのロジック）

相手の捨て牌の特定のパターンから、具体的な待ちを推測する技術です。

### 3.1 跨ぎスジ（Matagi-Suji）
「リーチ宣言牌」や「テンパイ直前の手出し牌」の周辺（跨ぎ）は危険という理論。

*   **理論**: 多くの場合、テンパイ直前まで両面変化などを期待して、将来の待ちに関連する牌を持っています。
*   **例**: 「5」を手出ししてリーチ → 「5」をまたぐ「3-6」や「4-7」待ちが本線（危険）となります。

### 3.2 裏スジ（Ura-Suji）
序盤に切られた牌の「裏側」に当たるスジが危険になる理論。

*   **基本**: 「5」が切られたら、「1-4」または「6-9」が危険。
*   **理由**: 「135」から5を切って1-3固定（2待ち）→その後変化して1-4待ちになる、などの手作り手順から導かれます。

### 3.3 間四ケン（Aida-Yon-Ken）
同色の数牌が「4つ飛ばし」で捨てられている場合、その間のエリア（内側）が非常に危険という理論。

*   **例**: 「3」と「8」が捨てられている場合（間が4、5、6、7の4つ）。
*   **読み**: 「3」や「8」はいらないが、「4567」周辺の牌で手がまとまっている可能性が高く、この「間」の牌が待ちになっている（あるいは危険牌である）ケースが多いです。

## 4. 状況判断と役読み (Context & Yaku)

### 4.1 状況別の読み (Contextual Reading)
読みは盤面のフェーズによって有効性が異なります。

| フェーズ | 読みの焦点 | セオリー |
| :--- | :--- | :--- |
| **序盤 (1-6巡目)** | **手役と速度** | 字牌や1/9牌の切り出し順を見る。役牌絞りは有効か？染め手の気配はあるか？ |
| **中盤 (7-12巡目)** | **ターツ構成** | 手出しされた中張牌周辺（跨ぎ）を警戒。安直なドラ切りはテンパイ濃厚サイン。 |
| **終盤 (13-18巡目)** | **危険牌一点読み** | 残りスジの本数を数える。テンパイノーテンの判断で押し引きを決める。 |

### 4.2 強者レベルの安全度判断
単なる「スジだから安全」ではなく、以下の優先度で厳密に守備を選択します。

1.  **完全安牌 (Genbutsu)**: リーチ者、同巡フリテンの牌。
2.  **ノーチャンス (Wall)**: 壁（4枚見え）で否定された牌。
3.  **合わせ打ちの中筋**: 上家が切り、リーチ者のスジでもある牌。
4.  **序盤切りの外側**: 早いリーチに対する、早い段階で切られた牌の外側（1/9）。
5.  **スジの1/9牌**: シャンポンや単騎に当たる確率は残るが、両面は否定される。
6.  **字牌の生牌**: **危険**。特にファン牌（役牌）の生牌は、スジよりも放銃率が高いことがあるため要注意。

### 4.3 役別の読み (Yaku-Specific Reading)
*   **染め手**: 余剰牌（溢れ）が出たらテンパイ濃厚。他色の雑な切り出しは染め手サイン。
*   **国士無双**: 序盤から中張牌がバラ切りされる。終盤に1・9・字牌が余らない、または1枚切れ字牌の手出しは要注意。
*   **トイトイ**: 2副露以上でトイツ場の気配（中張牌の手出し対子落とし等）。生牌が極めて危険。

### 4.4 副露読み (Naki-Yomi)
*   **食い伸ばし**: 「2」をチーして「5」切りなど、スライドや複合形からの変化を読む。
*   **役バック**: 役牌が見えていないのに数牌から鳴く場合、役牌バック（後付け）の可能性大。

## 5. デジタル・AI理論 (Digital & AI Theory)

現代の「科学する麻雀」やAI（Suphx, NAGA等）から導き出された高度な理論です。

### 5.1 統計データによる裏付け
*   **リーチへの危険度**: 無筋（10-15%） vs スジ（2-5%）。生牌字牌は無筋並みに危険（約10%）。
*   **先制リーチ**: 愚形でも先制なら即リーチが期待値的に有利。

### 5.2 AI視点の「読み」
*   **逆順切り**: 弱い牌より強い牌が先に切られる（例：1より3が先）＝手が整っている、または特定役狙い。
*   **ラグ読み (Latency Reading)**: ネット麻雀特有。鳴きボタン反応による「間」から、対子や塔子の存在を確定させる。

### 5.3 先知情報による学習 (Oracle Guiding)
AIは「全情報が見える状態」での学習を経て推論精度を高めています。人間も**「見えている情報（捨て牌・副露・点数）」だけを根拠**に論理的に手牌を限定すべきであり、直感に頼るべきではありません。

### 5.4 全局収支予測 (Global Reward Prediction)
「その局の勝負」ではなく「最終順位」を最大化する判断。
*   **トップ目**: 些細なリスクも極端に回避（過剰なオリが正解）。
*   **ラス目**: 多少の危険を無視してでも自身のアガリを最大化する。

## 6. 実践ドリル (Practical Drills)

*   **「何切る」ではなく「何持ってる」**: 捨て牌から相手の手牌構成（ブロック）を逆算する。
*   **ラグ読み**: ネット麻雀でのラグ牌をメモし、終局後に確認する。
*   **牌譜検討**: AI解析ツールを用いて、自分の放銃だけでなく相手の打牌理由を追求する。
